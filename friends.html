Demo hello.api('me/friends')

This is a test page for our adapted version of the HelloJS (http://adodson.com/hello.js/) client side Javascript SDK for making OAuth2 authenticated logins. Our version is an OAuth2 only adaptation, which is also streamlined for just our needs. Please note, this demo page will only run correctly as: https://opensitemobile2.github.io/hellojs/friends.html, as most services require it to run under SSL (https protocol).
